About

Like in baseball pitching, some songs are great for leading off a lullaby session ('openers'), some are for making it through the mid-game ('relievers'), and some you want to end with ('closers').

Provide a YouTube playlist ID for each of these categories and let this page randomly select an opener, 3 relievers, and a closer.

Example: mjkerrison.github.io/lullaby-bullpen/?openers=PL5kRBZR0MOfgY5TaFWUaI7MeLuCjDSqp5&relievers=PL5kRBZR0MOfiZPWyVXFc_dc8CxTFVKe6c&closers=PL5kRBZR0MOfhFqbYNB6aYl4ReTsvMCxQL

Enjoy bedtime singing, enjoy variety, and manage the cadence of your session - all at the same time.

(The app will autoplay like a real playlist!)
